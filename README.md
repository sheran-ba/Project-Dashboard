# Project Dashboard

A modern, interactive project management dashboard built with vanilla HTML, CSS, and JavaScript. Track your projects with an intuitive Kanban board, manage team members, and monitor progress all in one place.

## 🚀 Features

- **Kanban Board** - Organize projects across 4 columns: To Do, In Progress, Review, and Done
- **Drag & Drop** - Easily move projects between columns
- **Project Management** - Create, edit, and delete projects
- **Team Management** - Add team members and assign projects to them
- **Progress Tracking** - Visual progress bars for each project
- **Deadline Monitoring** - Automatic alerts for overdue and upcoming deadlines
- **Priority Levels** - Set Low, Medium, or High priority for each project
- **Search & Filter** - Quickly find projects by title or description
- **Persistent Storage** - All data is saved in your browser's localStorage
- **Overview Dashboard** - See team progress, completion rates, and upcoming deadlines at a glance
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices

## 🎯 Getting Started

### Live Demo
Visit the live dashboard here: [Project Dashboard](https://sheran-ba.github.io/Project-Dashboard/)

### Local Usage
1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start adding projects and managing your team!

## 📋 How to Use

### Creating a Project
1. Click the **"+ New Project"** button in the toolbar
2. Fill in the project details:
   - **Title** - Project name
   - **Description** - Notes about the project
   - **Owner** - Assign to a team member
   - **Status** - Choose To Do, In Progress, Review, or Done
   - **Deadline** - Set a due date
   - **Priority** - Set priority level
   - **Progress** - Set completion percentage (0-100%)
3. Click **Save Project**

### Moving Projects
- Click and drag projects between columns to change their status
- When you move a project to "Done", progress automatically sets to 100%

### Managing Team Members
1. Click **"Manage Team"** button
2. Enter a team member's name and click **Add**
3. View all team members with their assigned project count
4. Remove team members using the **Remove** button

### Overview Tab
- See total projects, active projects, and completed projects
- View team progress with completion rates
- Monitor upcoming deadlines
- Track average project progress

### Searching Projects
1. Use the **search box** in the toolbar
2. Type to filter projects by title or description
3. Results update in real-time

## 💾 Data Storage

All your data is automatically saved to your browser's **localStorage**. This means:
- ✅ Your data persists when you refresh the page
- ✅ Your data is stored locally on your device
- ✅ No server or account needed
- ⚠️ Data is browser-specific (clearing cache will delete data)
